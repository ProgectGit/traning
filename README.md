# Тренування

Односторінковий план тренувань українською мовою.

Можливості:

- таймер для підходів або відпочинку;
- загальний прогрес і окремий прогрес для кожного дня;
- звуковий сигнал після завершення таймера з підготовкою звуку для запуску з головного екрана;
- зручне введення часу з автоматичним очищенням нуля у полі;
- запуск з порожнього плану без заготовлених днів;
- додавання власних днів і вправ з окремими полями для підходів, повторень та ваги;
- редагування підходів, повторень і ваги у вже доданих вправах;
- збереження стану в `localStorage`;
- швидке обнулення відміток або очищення плану.

Структура:

- `index.html` - розмітка сторінки;
- `styles.css` - стилі та адаптивність;
- `app.js` - логіка таймера, вправ, прогресу і збереження.

Відкрийте `index.html` у браузері або опублікуйте репозиторій через GitHub Pages.
